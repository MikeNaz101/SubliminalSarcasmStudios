FP/TD Bugs

Play Now!

Description

'FP/TD Bugs' is a high-octane hybrid of a fast-paced, first-person shooter and a strategic tower defense game, built in Unity. The core loop challenges you to prevent the Robo-Bugs from reaching their destination by strategically building towers, all while fighting off relentless waves of them with your own arsenal. Some PLANET call for an exterminator?? That's you!

Project Status

The game is in active development. Many core systems are operational, and we're now focused on Tower logic, upgrades, a dynamic wave-spawning system, bug-fixing (the REAL EVIL ones), and feature polish.

What's Working:

  • Dynamic Player Controller: Fast, responsive, Doom-inspired FPS movement and combat.
  • Weapon & Recoil System: A hitscan weapon controller with ADS, camera shake for recoil, and a full reload system.
  • Tower Building: A grid-less building system where players spend 'Scrap' to place towers. Includes smart placement rules to prevent building on enemy paths or too close to other towers.
  • Advanced Enemy AI: Enemies use a NavMesh system to attack the Player, the Core, or even destructible 'Doors' to create new attack routes.
  • Wave Spawning: A robust Wave Manager that spawns different enemy types from Scriptable Object configurations.
  • Distraction & Vision: Enemies react to player-made sounds (gunshots) and use a Field of View component to acquire targets.
  • Adaptive Difficulty: An 'AI Director' monitors player accuracy. If you're too accurate, enemies start using evasive dodges. If you're struggling, they become more aggressive and predictable.

What's Next (Working Towards):

  • Expanding the arsenal with new weapons and tower types.
  • Designing more complex enemy types (like the 'Spawner' Queen).
  • Building and balancing a full set of levels.
  • Creating the final 3D models and sound design.
